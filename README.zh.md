# Agent Master Handbook

面向 agent 实践者的 AI Native 实践手册——既是学习资源，也是可部署的 agent-ready 基础设施。

以 AI Native 方式构建，项目本身即是方法论的实践。

## 这是什么

- **知识库**：核心概念（harness 工程、上下文工程、MCP）、实践指南、来自顶级 AI 工程博客的精选文章
- **基础设施**：skills、rules、specs 和模式，可被任何仓库采用以实现 agent-ready
- **网站**：从零构建的中英双语静态站点，优化信息展示和 agent 可解析性

## 快速开始

```bash
npm install
npm run build
# 打开 site/index.html
```
