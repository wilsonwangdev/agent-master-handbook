# 构建 AI Agent 的实践指南

**来源**：[OpenAI](https://openai.com/business/guides-and-resources/a-practical-guide-to-building-ai-agents/)

## 入选理由

OpenAI 的 agent 设计综合框架——涵盖用例识别、逻辑设计模式、编排和安全性。提供了一个厂商中立的 agent 架构思维模型。

## 核心要点

- 从用例识别开始：不是每个问题都需要 agent
- Agent 逻辑设计模式：路由、工具使用、规划和反思
- 多 agent 系统的编排模式
- 安全性和护栏作为一等关注点，而非事后补充
- Agent 可靠性的评估方法论

## 对实践者的价值

对 agent 工程新手来说是坚实的基础框架。以更广泛的架构视角补充了 Anthropic 和 Manus 更偏实现的具体指导。
