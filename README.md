# agent-master

An AI Native knowledge base for agent practitioners — both a learning resource and deployable infrastructure for making any agent environment agent-ready.

Built with AI Native practices. This project dogfoods its own methodology.

## What This Is

- **Knowledge base**: Core concepts (harness engineering, context engineering, MCP), practical guides, curated articles from leading AI engineering blogs
- **Infrastructure**: Skills, rules, specs, and patterns that can be adopted to make any repository agent-ready
- **Website**: A bilingual (EN/ZH) static site built from scratch, optimized for information display and agent parseability

## Quick Start

```bash
npm install
npm run build
# Open site/index.html
```
